# BarGraph - バーグラフアニメーション

64本のバーが正弦波パターンでアニメーションするカラフルなバーグラフ表示プログラムです。

## 📋 概要

このプログラムは、M5Stack Tab5の画面に64本のバーグラフを表示し、正弦波パターンで滑らかにアニメーションさせます。各バーはカラフルなグラデーションで表示され、視覚的に美しいエフェクトを提供します。

## 🔧 必要なハードウェア

- M5Stack Tab5

## 📚 必要なライブラリ

- **M5Unified**: M5Stack統合ライブラリ
  ```bash
  arduino-cli lib install "M5Unified@0.2.10"
  ```

## 🚀 セットアップ

1. **ボード設定**
   - Board: ESP32P4 Dev Module
   - USB CDC on boot: Enabled
   - Flash Size: 16MB (128Mb)
   - Partition Scheme: Custom
   - PSRAM: Enabled

2. **プログラムのアップロード**
   - Arduino IDEまたはArduino CLIでプログラムをアップロード

## 💻 使用方法

プログラムをアップロードすると、自動的にバーグラフアニメーションが開始されます。画面に64本のバーが表示され、正弦波パターンで滑らかに動きます。

## 📊 機能

- **64本のバーグラフ**: 画面に64本のバーを表示
- **正弦波アニメーション**: 各バーが正弦波パターンで動く
- **カラフルなグラデーション**: バーごとに異なる色で表示
- **滑らかなアニメーション**: フレームごとに滑らかに更新

## 🔍 プログラムの動作

1. **初期化**
   - M5Stackの初期化
   - 画面の設定（回転、EPDモードなど）
   - 各バーの初期位置と色の設定

2. **アニメーションループ**
   - 各バーの位置を正弦波で計算
   - 前の位置をクリア
   - 新しい位置にバーを描画
   - 画面を更新

## 📝 コードの主要なポイント

### バーの数
```cpp
static constexpr size_t BAR_COUNT = 64;  // 64本のバー
```

### 色の設定
各バーは16本ごとに異なる色グループに分類され、グラデーションで表示されます。

### アニメーション
正弦波関数を使用して各バーの高さを計算し、滑らかなアニメーションを実現しています。
