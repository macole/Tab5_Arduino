# M5Stack Tab5 サンプルプログラム集

このリポジトリは、M5Stack Tab5向けのArduinoサンプルプログラム集です。

## 概要

M5Stack Tab5の各種機能を試すためのサンプルプログラムを提供しています。各サンプルは独立して動作し、M5Unifiedライブラリを使用しています。

## 必要な環境

- Arduino IDE または PlatformIO
- [M5Unified](https://github.com/m5stack/M5Unified) ライブラリ
- M5Stack Tab5

## サンプルプログラム一覧

### BarGraph
バーグラフのアニメーション表示サンプルです。64本のバーが波のように動くビジュアルエフェクトを実装しています。

**主な機能:**
- 64本のバーグラフ表示
- グラデーションカラー
- スムーズなアニメーション

### DragDrop
タッチスクリーンを使ったドラッグ&ドロップ操作のサンプルです。画面の4隅に配置された色付きボックスをタッチしてドラッグできます。

**主な機能:**
- マルチタッチ対応
- ドラッグ&ドロップ操作
- タッチ状態の検出（touch, hold, flick, drag）

### IMU
IMU（慣性計測装置）センサーのリアルタイムデータ表示サンプルです。加速度とジャイロの値を画面に表示します。

**主な機能:**
- 加速度センサー（X, Y, Z軸）の値表示
- ジャイロセンサー（X, Y, Z軸）の値表示
- リアルタイム更新

### IMU02
IMUセンサーを使ったボール制御ゲームの基本版です。デバイスを傾けると画面のボールが動きます。

**主な機能:**
- IMUデータによるボール制御
- 物理演算（速度、減衰、反発）
- 画面端での衝突判定

### IMU03
IMUセンサーを使ったボール制御ゲームの最適化版です。スプライトを使用して描画を最適化し、衝突時に音が鳴ります。

**主な機能:**
- IMUデータによるボール制御
- スプライトによる高速描画
- 衝突時の音声フィードバック
- HUD表示の最適化

### IMU04
IMUセンサーを使ったボール制御ゲームの別実装版です。全画面スプライトを使用した実装です。

**主な機能:**
- IMUデータによるボール制御
- 全画面スプライトバッファ
- チラつきのない描画

### MIC
マイクを使った録音・再生サンプルです。マイクで録音した音声をスピーカーから再生できます。

**主な機能:**
- 音声の録音（最大256フレーム）
- 波形のリアルタイム表示
- タッチで再生
- サンプリングレート: 17kHz

### Power
バッテリー情報を表示するサンプルです。充電状態、電圧、残量を表示します。

**主な機能:**
- 充電状態の表示
- バッテリー電圧の表示（mV）
- バッテリーレベルの表示（%）

### RTC
リアルタイムクロック（RTC）のサンプルです。日時を表示し、設定も可能です。

**主な機能:**
- RTCの日時表示
- 日時の設定
- 曜日の表示

### SD
SDカードの読み書きサンプルです。テキストファイルの書き込みとPNG画像の表示を行います。

**主な機能:**
- SDカードの検出
- テキストファイルの書き込み
- PNG画像ファイルの読み込みと表示
- SPI通信によるSDカードアクセス

### Speaker
スピーカーのテストサンプルです。異なる周波数の音を順番に再生します。

**主な機能:**
- 複数の周波数で音声再生
- トーン生成（2000Hz～10000Hz）

## 使い方

1. このリポジトリをクローンまたはダウンロードします
2. Arduino IDEを開き、M5Unifiedライブラリをインストールします
3. 使用したいサンプルフォルダを開きます
4. `.ino`ファイルをArduino IDEで開きます
5. M5Stack Tab5をUSBケーブルで接続します
6. ボードを選択し、アップロードします

## ライセンス

このサンプルプログラムは、M5Stack Tab5の機能を学習・実験するためのものです。
