# GameOfLife - ライフゲーム

Conway's Game of Life（ライフゲーム）を実装したプログラムです。

## 📋 概要

このプログラムは、1970年にイギリスの数学者ジョン・ホートン・コンウェイによって考案されたセルオートマトン「ライフゲーム」を実装しています。セル（細胞）の生死のルールに基づいて、世代ごとにパターンが進化していきます。

## 🔧 必要なハードウェア

- M5Stack Tab5

## 📚 必要なライブラリ

- **M5GFX**: M5Stack用グラフィックライブラリ
  ```bash
  arduino-cli lib install "M5GFX@0.2.15"
  ```

## 🚀 セットアップ

1. **ボード設定**
   - Board: ESP32P4 Dev Module
   - USB CDC on boot: Enabled
   - Flash Size: 16MB (128Mb)
   - Partition Scheme: Custom
   - PSRAM: Enabled

2. **プログラムのアップロード**
   - Arduino IDEまたはArduino CLIでプログラムをアップロード

## 💻 使用方法

プログラムをアップロードすると、自動的にライフゲームが開始されます。画面にセルのパターンが表示され、世代ごとに進化していきます。

## 📊 機能

- **セルオートマトン**: ライフゲームのルールを実装
- **リアルタイム更新**: 世代ごとにパターンが更新される
- **画面サイズ自動調整**: 画面サイズに応じてセルサイズを自動調整
- **ダブルバッファリング**: スムーズな表示のためのダブルバッファリング

## 🎮 ライフゲームのルール

1. **生存**: 生きているセルが2個または3個の隣接セルを持つ場合、次の世代も生存
2. **死亡**: 生きているセルが1個以下または4個以上の隣接セルを持つ場合、次の世代で死亡
3. **誕生**: 死んでいるセルがちょうど3個の隣接セルを持つ場合、次の世代で誕生

## 🔍 プログラムの動作

1. **初期化**
   - 画面の初期化
   - セルサイズの計算（画面サイズに応じて自動調整）
   - ダブルバッファの作成

2. **ゲームループ**
   - 現在の世代のセル状態を確認
   - ライフゲームのルールに基づいて次の世代を計算
   - ダブルバッファを使用して画面を更新

## 📝 コードの主要なポイント

### セルサイズの自動調整
画面サイズに応じて、最適なセルサイズを自動計算します。

### ダブルバッファリング
2つのキャンバスを使用して、スムーズな画面更新を実現しています。
